Turkish B2 Writing Simulator (Sevim Hoca) ✍️🇹🇷

🇺🇦 Українською 

Turkish B2 Writing Simulator — це інтерактивний веб-додаток, розроблений для допомоги студентам, які вивчають турецьку мову на рівні B2, у написанні офіційних звітів (Rapor).

Проєкт включає віртуального помічника — Sevim Hoca, яка аналізує ваш текст у реальному часі, виставляє бали та надає персоналізовані поради щодо покращення академічного стилю.

✨ Особливості

Аналіз структури: перевірка всіх частин звіту (Giriş, İçerik, Sonuç тощо).

Зворотний зв'язок: поради щодо використання специфічних сполучників та академічної лексики.

Оцінювання: система балів на основі критеріїв рівня B2.

Темна / світла теми: зручний інтерфейс для роботи в будь-який час.

🇬🇧 English Translation

Turkish B2 Writing Simulator is an interactive web application designed to help learners of Turkish at the B2 level write formal reports (Rapor).

The project features a virtual assistant — Sevim Hoca — who analyzes your text in real time, assigns scores, and provides personalized feedback to improve your academic writing style.

✨ Features

Structure analysis: checks all parts of a report (Introduction, Main Body, Conclusion, etc.).

Feedback: guidance on using academic connectors and formal vocabulary.

Assessment: scoring system based on B2-level criteria.

Dark / Light modes: a comfortable interface for any time of day.

📚 Materials

The repository also includes a vocabulary file:

Yuliia Saçkesen – vocab_rapor.docx — a thematic vocabulary list for report writing based on materials from Yunus Emre Institute.

🚀 How to Use

Open [your GitHub Pages link].

Enter the report details and its purpose.

Write your text in the provided fields using Sevim Hoca’s prompts.

Click “Raporu Gönder” to receive the analysis.

🇹🇷 Türkçe Çeviri

Turkish B2 Writing Simulator, B2 seviyesinde Türkçe öğrenen öğrencilerin resmî rapor (Rapor) yazma becerilerini geliştirmek amacıyla hazırlanmıştır.

Projede yer alan sanal asistan Sevim Hoca, metninizi gerçek zamanlı olarak analiz eder, puanlar ve akademik yazım stilinizi geliştirmek için kişiselleştirilmiş geri bildirimler sunar.

✨ Özellikler

Yapı analizi: raporun tüm bölümlerinin kontrolü (Giriş, İçerik, Sonuç vb.).

Geri bildirim: akademik bağlaçlar ve resmî kelime kullanımıyla ilgili öneriler.

Değerlendirme: B2 seviye kriterlerine dayalı puanlama sistemi.

Koyu / açık tema: her zaman dilimi için rahat bir arayüz.
